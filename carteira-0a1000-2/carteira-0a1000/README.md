# 0 a 1000 — Painel de Investimentos

Painel pessoal de acompanhamento de carteira de investimentos.

## Funcionalidades

- Resumo da carteira com métricas principais
- Análise por região, sector e tese de investimento
- Lista de empresas com links directos aos relatórios ao investidor (IR)
- ETFs e instrumentos estruturais com fichas de produto
- Registo de aportes mensais com gráfico acumulado
- Calendário de relatórios ao investidor com integração Google Calendar
- Cotações via Google Sheets + GOOGLEFINANCE

## Como usar

Acede em: **https://[o-teu-username].github.io/carteira-0a1000**

## Cotações

Para activar as cotações em tempo real:
1. Importa o ficheiro `carteira_0a1000_googlesheets.xlsx` para o Google Sheets
2. Partilha a folha como pública (visualizador)
3. Cola o ID da folha na aba Cotações do painel

---
*Painel pessoal — não constitui aconselhamento financeiro.*
